Tables in Georev

gr_v2.gr_country_region

This table contains mappings from ISO_COUNTRY codes to the region structure.

  • iso_country: Two-letter code representing a country
  • start_date: Date the record was first published
  • end_date: Date the record was terminated. A null end_date indicate the record is considered current.
  • parent_region_id: Identifier representing the geographic area that the country is a part of
  • path: Fully articulated path to this geography from the geographic tree's root sector, separated by > signs. Each layer in the path is indicated by its Name value in this table.
  • effective_start_date: Date that the country-to-region mapping was first considered effective
  • effective_end_date: Date that the country-to-region mapping was no longer considered effective. The date will reflect when items will begin to be assigned to the new region based on the period end date of the report.
  • country_name: Name of the country that corresponds with the ISO_COUNTRY code between the effective start and end dates
gr_v2.gr_coverage

This table contains coverage information for each entity that has a report in the Geographic Revenue Exposure package.

  • factset_entity_id: Unique FactSet-generated identifier representing an entity
  • actively_covered: Flag indicating whether data is actively being collected for the company
  • latest_period_end_date: Date of the last report period collected
gr_v2.gr_item

This table contains revenue exposures at the country level for all records, current and historical.

  • report_id: Unique identifier for the report. The report is the container entity for all items detailed by the company in a single period, as described in this record.
  • iso_country: Two-letter code representing the country for which geographic revenue data is being reported
  • start_date: Date the record was first published
  • end_date: Date the record was terminated. A null end_date indicate the record is considered current.
  • est_pct: Estimated percentage of revenue derived from the associated country
  • conf: Confidence score for the value in Est_Percent and Est_Amount fields. A Conf score of 1 indicates a declared value, which automatically has a 100% confidence, or a calculated total value, which has a logic confidence of 100%. A Conf score of 0.50 to 0.99 indicates an extrapolated value, with progressively lower scores denoting decreasing certainty of precision.
  • certainty_rank: Certainty score for the value in Est_Percent based on source metadata. Ranges from 1 (low certainty) to 80 (declared value).
  • certainty_class: A grouping for the value in Est_Pct based on Certainty Rank. Ranges from E (low) to A (high).

  • factset_entity_id: Unique FactSet-generated identifier representing an entity
  • factset_ult_parent_entity_id: Unique FactSet-generated identifier representing the ultimate owner of the entity's equity
gr_v2.gr_region_structure

This table contains the reference region and geographic tree data for all geographic exposure tree reports.

  • region_id: Identifier representing the geographic region
  • start_date: Date the record was first published
  • end_date: Date the record was terminated. A null end_date indicate the record is considered current.
  • parent_id: Unique identifier representing the geographic region which is the immediate parent to this region
  • name: Name of the geographic region
  • layer_number: Code denoting the depth of region's location within the geographic tree:0 = Global, 1 = Super-Region, 2 = Region, 3 = Area, and 4 = Country
  • path: Fully articulated path to this geography from the geographic tree's root sector, separated by > signs. Each layer in the path is indicated by its Name value in this table.
gr_v2.gr_report

This table contains the information about the reporting periods for entities with Geographic Revenue Exposure data.

  • report_id: Unique identifier for the report. The report is the container entity for all items detailed by the company in a single period, as described in this record.
  • start_date: Date the record was first published

  • period_end_date: Date on which the report period completed. This is usually, but not always, the end of a fiscal period.

  • fax_full: Full fax number, including the country code and area code

  • location_street2: Second street line in the entity's address
  • location_street3: Third street line in the entity's address
  • iso_country: Two-letter code representing a country
  • tele_country: Country code of the telephone number
  • tele_area: City or area code of the telephone number
